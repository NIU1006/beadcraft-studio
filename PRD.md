# 豆趣工坊 (BeadCraft Studio) - 产品需求文档

## 1. 产品概述

### 1.1 产品定位
豆趣工坊是一款专为拼豆爱好者设计的在线图纸生成工具，通过智能配色算法将用户上传的图片转换为可制作的拼豆图纸，支持多品牌色卡对照和材料统计。

### 1.2 目标用户
- 拼豆手工爱好者
- 像素艺术创作者
- DIY手工博主
- 需要快速生成拼豆图纸的商家

### 1.3 核心价值
- **智能配色**：自动匹配最接近的拼豆颜色，无需手动比对色卡
- **多品牌支持**：覆盖市面上主流拼豆品牌，方便用户按需选择
- **一键出图**：上传图片即可生成带颜色标号的制作图纸
- **材料清单**：自动统计各色豆子用量，方便采购准备

## 3. 技术规格

### 3.1 技术选型
| 模块 | 技术方案 | 说明 |
|------|----------|------|
| 前端框架 | React + TypeScript | 组件化开发，类型安全 |
| 状态管理 | Zustand | 轻量级状态管理 |
| 样式方案 | Tailwind CSS | 快速响应式布局 |
| 图片处理 | Canvas API | 前端像素处理 |
| 颜色计算 | 自定义 CIE76 算法 | LAB 颜色空间色差计算 |
| 图表展示 | ECharts | 材料统计可视化 |
| PDF 生成 | jsPDF + html2canvas | 客户端 PDF 导出 |

### 3.2 颜色匹配算法
```
1. 将色卡颜色从 RGB 转换为 LAB 颜色空间
2. 对图片每个像素进行同样的转换
3. 计算与所有色卡颜色的 CIE76 色差（Delta E）
4. 选择 Delta E 最小的颜色作为匹配结果
```

### 3.3 数据结构

**色卡数据格式：**
```typescript
interface BeadColor {
  id: string;           // 唯一标识
  brand: string;        // 品牌名称
  code: string;         // 色号
  name: string;         // 颜色名称
  rgb: [number, number, number];  // RGB值
  lab: [number, number, number];  // 预计算LAB值
}
```

**图纸数据格式：**
```typescript
interface PatternData {
  width: number;        // 画板宽度
  height: number;       // 画板高度
  pixelSize: number;    // 像素尺寸(mm)
  brand: string;        // 使用品牌
  pixels: BeadColor[][]; // 二维颜色数组
  statistics: {
    colorId: string;
    count: number;
  }[];
}
```

## 4. 界面设计

### 4.1 页面结构
```
┌─────────────────────────────────────────────────────────────┐
│  顶部导航栏 (Logo + 品牌选择 + 使用帮助)                      │
```

- 颜色块 + 色号 + 名称 + 数量
- 复制清单按钮
- 展开/收起控制

## 5. 非功能需求

### 5.1 性能要求
- 图片处理响应时间 < 2秒（50x50尺寸）
- 页面首屏加载 < 3秒
- 支持最大 200x200 画板尺寸处理

### 5.2 兼容性
- 支持 Chrome、Firefox、Safari、Edge 最新两版
- 移动端浏览器基础功能可用
- 响应式布局适配平板和桌面

### 5.3 可访问性
- 支持键盘操作
- 颜色对比度符合 WCAG 2.1 AA 标准
- 提供色盲友好模式（图案区分）
